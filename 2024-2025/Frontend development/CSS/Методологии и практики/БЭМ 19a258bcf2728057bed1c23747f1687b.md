# БЭМ

**Б - Блок (отделимая составляющая сайта)**

**Э - Элемент блока (составная часть блока т.е. часть блока которая не может использоваться в вакууме (например подпись под картинкой) и это элемент)**

**М - Модификатор(то что задает элементу внешний вид)**
