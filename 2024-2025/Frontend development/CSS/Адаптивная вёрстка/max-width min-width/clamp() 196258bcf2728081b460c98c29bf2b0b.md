# clamp()

Что то между min()/max(). Только тут можно добавить ещё одно значение. То есть так clamp(min, x, max) на самом деле очень удобно для адаптива изображений + это правило ещё и с размером текста юзают. 

Ну или как сказано на доке минимальное значение(при нижней границе допустимого)/предпочтительное/максимальная(при верхней границе допустимого)

```css
.box {
  width: clamp(240px, 80%, 360px);
  margin: 0 auto;
}
```
