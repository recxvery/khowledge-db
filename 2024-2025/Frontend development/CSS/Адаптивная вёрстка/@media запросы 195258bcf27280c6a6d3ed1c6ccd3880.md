# @media запросы

Это очень похоже на функции и по синтаксису и в целом. То есть типо когда у нас меняется разрешение экрана, то стили меняют своё поведение. достаточно интересно в прочем. Хотя по началу пиздец эта хуйня бесила сука, хотелось уебать её нахуй. Короче, видимо норм взаимодействует с флексами ещё. 

Синтаксис: 

```css
@media (min-width: 1024px) {
  html {
    background-color: #27388b;
    color: white;
    font-size: 24px;
  }
}

@media (max-width: 1024px) and (min-width: 768px) {
  html {
    background-color: green;
    color: black;
    font-size: 20px;
  }
}

@media (max-width: 768px) {
  html {
    background-color: yellow;
    color: black;
    font-size: 16px;
    text-align: center;
  }
}
```

```css
@media (hover: hover) {
  #heroButton:hover {
    background-color: #7d8bd1;
  }
}

```
