# max-width | min-width

Те самые два брата красавчика из тик тока. Ладно на самом деле тут всё изи. как только значение доходит до этих параметров что-то происходит. например при уменьшении экрана если вьюпорт доходит до min-width: 360px и ниже то странице начинается уменьшаться вся. А не например граница кнопок и полей.
