# calc()

Короче эта имбулся я так понял позволяет нам воспроизводить рассчёты внутри CSS файла непосредственно. А динозавры держу в курсе писали 

```css
width: 33.3333333333333333333px
```

Щас всё намного проще и делается это вот так: 

```css
width: calc(100% / 3);
```

Так что calc ещё имба, если мы например юаем опредленные переменные которые потом  нам необходимо будет изменить. Короче стоящая штука, хоть и сложная. 

Нельзя писать вот так: 

![image.png](2024-2025/Frontend%20development/CSS/Адаптивная%20вёрстка/calc()/image.png)

И нельзя делить вот так вот например:

```css
width: calc(100% / 300px) // ширина должна быть одинаковой всегда.
```
