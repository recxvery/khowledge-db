# min() / max()

Имба. короче позволяет регулировать условный размер предмета возвравщая наименьшее значение из фукнции.

```css
width: min(200px, 100% / 3)
// пока размер страницы больше 600 ширина предмета = 200px, 
//то есть функция выбирает из меньших
```

Для max() всё то же самое только наоборот. Выносим максимальное значение из функции:

```css
width: max(400px, 100% / 3)
```
