# flex-direction

отвечает за расположение элементов. по горизонтали или вертикали + реверсии.

# Параметр order:

отвечает за расположение элемента в блоке флекс хз как объяснить ещё. по умолчанию значение - order:1. то есть у всех элементов значение единица. Потому если поставим значение 5 то элемент будет в конце хоть будь там 100 элементьов. или такуже если поставим -1 то будет в самом начале.

## flex-wrap

Свойство flex-wrap определяет, будут ли элементы принудительно находиться в одной строке или могут быть перенесены на несколько строк. По умолчанию элементы пытаются разместиться в одной строке.

## nowrap (по умолчанию)

Все элементы пытаются разместиться в одной строке

## wrap

Элементы будут переноситься на новую строку, если не помещаются в одной

## wrap-reverse

Элементы переносятся на новую строку в обратном порядке

## flex-basis

вот из-за флекс врапа у нас переносятся элементы на другую строку условно. то флекс базис определяет их размер. например каждые 3 элемента будет новый ряд блоков тогда: 

```css
flex-basis: calc(100% / 3)
```

## flex-shrink

по умолчанию по моему он везде равен 1. но по факту если по ставим у определённого элемента 2. то он будет сжиматься быстрее других. грубо говоря указываем пропорции сжатия элементов. (ебать умно спизданул). Ну вообще наебал это грубо говоря кэф сжатия одного элемента по отношению к другим
