# Доступность (aria-атрибуты)

Доступность грубо говоря позволяет юзерам с ограниченными возможностяит взаимодействовать с сайтом. (в том числе и те у кого медленный инет и кто просто пидорас и кто с телефона сидит)

Для этого мы используем label’ы, правильную семантическую разметку, ариа-атрибуты(вот допустим человек слепой жмёт на кнопку, у него нахуй все бабки сняли - ну это здорово коненчо если мы владельцы сайта, но всё же. нам необходимо подписывать ээлементы таким образом допустим 

aria-expanded - означает свернута кнопка или нет

aria-haspopup - есть ли поп ап(то что поверх всех элементов) в данном случае есть кнопка бургер так что да

ещё есть aria-hidden она просто используется для скрытия label’a и описания элемента

```html
        <div class="header__wrapper">
            <a href=""><img class="header__logo" src="img/logo.svg" alt="Логотип PurpleSchool"></a>
            <nav>
                <ul class="menu">
                    <li class="menu__item menu__item_active"><a href="#">Курсы</a></li>
                    <li class="menu__item"><a href="#">О школе</a></li>
                    <li class="menu__item"><a href="#">Блог</a></li>
                    <li class="menu__item"><a href="#">База знаний</a></li>
                </ul>
            </nav>
            <a class="header__login " href="#" aria-label="Вход в личный кабинет">
                <img src="img/user.svg" alt="Иконка пользователя">
                <div>Вход</div>
            </a>
            <button class="header__mobile-menu-button" aria-expanded="false" aria-haspopup="true">
                <img src="img/menu.svg" alt="Мобильное меню">
            </button>
        </div>
```

# 5 правил доступности:

- Не забываем про теги семантики по типу header,nav,menu, footer, aside(юзается для боковых панелей) ещё есть какой-то
- Заголовок = заголовок(в видосе у чувака пиздец бомбит что все делают через div’ы)
- То же самое про <a> и <button> ни через какие div’ы или js.
- Не выключать outline без просьбы коллег выше по цеху.
- Ну и не забывать про aria | label
