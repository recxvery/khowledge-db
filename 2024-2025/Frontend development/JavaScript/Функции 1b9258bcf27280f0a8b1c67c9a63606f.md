# Сигнатура функции.

Можно сказать некоторая инструкция, что определяет как входные значения и их тип данных, так и выходные значения с их типом данных

```jsx
Math.pow(base, exponent)

Параметры
    base
        Основание степени.
    exponent
        Показатель степени, в которую возводится основание base.
```

Это сигнатура функции и короткое пояснение на русском языке.

### Если передать пустую функцию(без параметров) то выведется NaN.

# Детерминированность функции

Это когда функция для одних и тех же значений выдаёт одинаковый результат. То есть например:  

```jsx
length('hexlet'); // 6
length('wow'); // 3
```

### Пример недетерминированной функции:

```jsx
Math.random(); // 0.0985661311319767
Math.random(); // 0.8839904367241888
```

## Побочные эффекты в функциях:

Так ну это штуки которые типо как-то влияют на внешнее окружение или среду выполнения. К примеру перезаписывают какой-либо файл, выводят какое-либо значение на экран ну или как по вики: 

- доступ (чтение или запись) к объекту, определённому с модификатором [volatile](https://en.wikipedia.org/wiki/volatile_(computer_programming)) ;
    
    (англ.)
    
- изменение (запись) объекта;
- изменение файла;
- изменение поведения инструкций [процессора](https://ru.wikipedia.org/wiki/%D0%9F%D1%80%D0%BE%D1%86%D0%B5%D1%81%D1%81%D0%BE%D1%80), обрабатывающих [числа с плавающей точкой](https://ru.wikipedia.org/wiki/%D0%A7%D0%B8%D1%81%D0%BB%D0%BE_%D1%81_%D0%BF%D0%BB%D0%B0%D0%B2%D0%B0%D1%8E%D1%89%D0%B5%D0%B9_%D0%B7%D0%B0%D0%BF%D1%8F%D1%82%D0%BE%D0%B9) (см. [floating-point environment](https://en.wikipedia.org/wiki/C_mathematical_functions#Floating_point_environment) );
    
    (англ.)
    
- вызов функции, выполняющей любое из перечисленных выше действий.
