# Валидация

![Untitled](2024-2025/Frontend%20development/HTML/Формы/Валидация/Untitled.png)

**Валидация** - когда система указывает на неправильный ввод email’a,пароль имеет некорректную длину и прочего.

 Валидация на FrontEnd’e нужна для того чтобы лишний раз не отправлять некорректные данные на Backend

```html
    <div>
        <label for="search">Введите запрос</label>
        <input 
        type="text"
        placeholder="Запрос" 
        id="search" 
        name="search_query"
        minlength="5"
        maxlength="20"
        pattern="^[ 0-9]+$"
        title="Нужно ввести числовой запрос"
        required>
        <input type="email" name="email" id="">
        <input type="url" name="url" id="">
    </div>
```

required - обязательное заполнение поля. Pattern - в данном случае нужно ввести цифры. minlength,maxlength - максимальный размер текста. в <input type=”email”>
