# label

![Untitled](2024-2025/Frontend%20development/HTML/Формы/Untitled%203.png)

Элемент **`<label>`** в HTML используется для создания метки (label) для элементов управления формой, таких как **`<input>`**, **`<textarea>`**, **`<select>`** и других. Основное предназначение элемента **`<label>`** - связать текстовую метку с соответствующим элементом формы. Вот несколько причин, по которым **`<label>`** полезен:

1. **Улучшение доступности**: Метки делают формы более доступными для пользователей, особенно для тех, кто использует программы чтения с экрана или другие адаптивные технологии. Прикрепление меток к элементам формы позволяет пользователям с ограниченными возможностями лучше понимать, что ожидается от каждого поля.
2. **Увеличение области щелчка**: Метка расширяет область, по которой можно щелкнуть для активации элемента формы. Это делает ввод данных более удобным для пользователей.
3. **Улучшение семантики**: Использование меток помогает улучшить семантику HTML, делая код более понятным и легко читаемым для разработчиков и поисковых систем.

# Checkbox,radio,select.

**Чек бокс:** 

![Untitled](2024-2025/Frontend%20development/HTML/Формы/Untitled%209.png)

```html
    <div>
        <label for="agree">Согласие на обработку персональных данных</label>
        <input type="checkbox" id="agree" name="agreement" checked> <!--сhecked - chekbox заполнен по умолчанию-->
    </div>
```

**Radio:**

```html
        <input type="radio" name="search" id="all" value="all">
        <label for="all">Поиск повсюду</label> <!-- из-за одинакого параметра name у нас нельзя поставить сразу два radio-->
    </div>
    <div>
        <input type="radio" name="search" id="yt" value="youtube">
        <label for="yt">Поиск на Ютубе</label>
```

**Select**

Code:

```html
       <label for="topic">Тема</label>
        <select name="topic" id="topic">
            <option value="NewsRussia" selected>Российские новости</option>
            <optgroup label="Популярные">
                <option value="war_ukraine">Война в Украине</option>
                <option value="terror">Теракт в Москве</option>
            </optgroup>
            <optgroup label="Киберспорт">
                <option value="csmajor">Copenhagen Major</option>
                <option value="dreamlegue_dota2_23">Dreamleague S23</option>
            </optgroup>
        </select>
```

Параметр selected указывает на то что будет выбрано по умолчанию. `Optgroup` - позволяет группировать необходимые нам запросы. Вот как это будет выглядеть.
