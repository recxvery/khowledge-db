![Untitled](2024-2025/EГЭ/ЕГЭ%20по%20информатике/13/Untitled%201.png)

всего нулей - 11. следовательно 2^11 = 2048. но из-за примечания 2048 - 2 = 2046. без примечания 2048

Задание в ручную:

![Untitled](2024-2025/EГЭ/ЕГЭ%20по%20информатике/13/Untitled%203.png)

```python
from ipaddress import *

net = ip_network('192.168.32.160/255.255.255.240')
for ip in net:
    s = f'{ip:b}'
    if s.count('1') % 2 == 0:
        c += 1
print(c)
```
