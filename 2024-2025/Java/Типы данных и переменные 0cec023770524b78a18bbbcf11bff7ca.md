char - это бессимвольный short. пишется в одинарных кавычках. строки в двойных. потому и говорится,что строка это набор символов.

**boolean - либо 1 либо 0 байт.**

Чтобы создать переменную - **Ctrl + alt + v. Объявляется она следующим образом:**

```java
public class lesson2example {

    public static void main(String[] args) {
        int x = 26;
        System.out.println(x);
    }
}
```

Чтобы изменять значение переменной не нужно указывать её тип данных.

```java
        char valueTest = 'a';
        System.out.println(valueTest);

        valueTest = 'x';
        System.out.println(valueTest);
```

Чтобы сделать переменную неизменяемойw(КОНСТАНТОЙ):

```java
        final char valueTest = 'a';
        System.out.println(valueTest);
```

```java
        int value = 10;

```

То есть при двух примерах кода результат будет отличаться. В первом случае 1,во втором 2:

```java
    public static void main(String[] args) {
    int x = 3/2;
        System.out.println(x++);
    }
```

```java
    public static void main(String[] args) {
    int x = 3/2;
        System.out.println(++x);
    }
```

Инверсия переменной с булевым значением (result = false):

```java
        int value = 10;
        boolean test = value != 11;
        System.out.println(!test);
```

Операторы сравнения в Java:

**И**

```java
        int value = 10;
        int value2 = -10;
        boolean test = -10 < 0;
        boolean test2 =  value > value2;
        System.out.println(test && test2);
```

**ИЛИ** 

```java
        int value = 10;
        int value2 = -10;
        boolean test = -10 < 0;
        boolean test2 =  value2 > value;
        System.out.println(test || test2);
```
