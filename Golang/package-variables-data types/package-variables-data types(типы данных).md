## Variables

  

> _**В GO нельзя оставлять неиспользованные переменные, импорты пакетов**_

  

Дефолт объявление переменной(названия по дефолту с CamelCase) :

```Go
func main() {
	var userheight = 1.92
	var userWeight float64 = 70
	var IMT = userWeight / math.Pow(userheight, 2)	
	fmt.Println(IMT)
}
```

![[мусор/image 7.png]]

  

Вот кстати, что интересно так это то что при объявлении переменных они сами же инициализируются то есть мы можем и не присваивать значение переменной,она все равно будет существовать и ей можно будет воспользоваться:

![[мусор/image 8.png]]

  

Также есть ещё один способ. Он очень похож на оператор присваивания в Python:

```Go
a := 5
//var a int = 5 == a := 5
```

  

  

Также можно объявить сразу несколько переменных в одном блоке **var**:

```Go
package main


import "fmt"
 
func main() {
    var (
        name string = "Dima"
        age int = 23
    )
     
    fmt.Println(name)
    fmt.Println(age)
}
```

  

Есть и привычные константы из JS

```Go
package main

import (
	"fmt"
	"math"
)

func main() {
	const userPower = 2 //в данном случае это нетипизированная константа,что значит 
	//что в строке ниже userPower будет уже float
	var userHeight, userWeight float64 = 1.92, 70
	IMT := userWeight / math.Pow(userHeight, userPower)	
	fmt.Println(IMT)
}
```
