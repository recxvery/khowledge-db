Базовый **break**

```Go
func main() {
	for i := 0; i < 10; i++ {
		if i == 5 {
			continue
		}
		fmt.Printf("%d\n", i)
	}
```

Базовый **continue**

```Go
func main() {
	for i := 0; i < 10; i++ {
		if i == 5 {
			continue
		}
		fmt.Printf("%d\n", i)

```
