Дефолтный IF/ELSE

```Go
	if IMT < 16 {
		fmt.Println("У вас критический дефицит массы тела")
	} else if 16 <= IMT && IMT < 18.5 {
		fmt.Println("У вас дефицит массы тела")
	} else if 18.5 <= IMT && IMT < 25 {
		fmt.Println("У вас нормальный вес")
	} else if 25 <= IMT && IMT < 18.5 {
		fmt.Println("У вас избыточный вес")
	} else {
		fmt.Println("У вас степень ожирения")
	}
```

_**единственное, что прям интересно, что “{” и else должны быть на одной строк**_

  

Switch конструкция:

```Go
	switch {
	case IMT < 16:
		fmt.Println("У вас критический дефицит массы тела")
	case IMT < 18.5:
		fmt.Println("У вас дефицит массы тела")
	case IMT < 25:
		fmt.Println("У вас нормальный вес")
	case IMT < 30:
		fmt.Println("У вас избыточный вес")
	default:
		fmt.Println("У вас степень ожирения")
	}
```
